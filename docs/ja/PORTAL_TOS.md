# QUBO++ User Portal — アカウント利用規約

**発効日:** 2026-05-09

本規約は、QUBO++ User Portal (<https://qubo-plus.github.io/portal/>) を通じ
て作成された QUBO++ ユーザーアカウント（以下「**アカウント**」）、および
アカウントに紐づく QUBO++ ライセンスキーの利用に適用されます。本規約は、
QUBO++ ソフトウェア全般に適用される [QUBO++ 利用規約](TERMS.html) を補足
するものです。

## 1. 同意
アカウントを作成した時点、またはアカウントの下で発行された任意のライセンス
キーをアクティベートした時点で、本規約に同意したものとみなします。同意でき
ない場合は、アカウントを作成しないでください。

## 2. アカウントについて
QUBO++ ユーザーアカウントは、ライセンスキーを個人に紐づけるための個人識別
情報です。アカウント作成時には自動的に **Trial ライセンス** が 1 本付与され
ます。当方は当方の裁量により、Unlimited ライセンスや別途取り決めた有償
ライセンス等、追加のライセンスをアカウントに紐づけることがあります。種別に
かかわらず、アカウントに紐づくすべてのライセンスは、本規約および各ライセン
ス発行時に通知される個別条件の双方に従います。

## 3. ライセンス許諾
アカウントに紐づく各ライセンスについて、当方は、ポータルのダッシュボードに
表示される期間・変数数上限・その他の条件の範囲で、QUBO++ を利用するための
限定的・非独占的・譲渡不能・取消可能なライセンスを許諾します。

Trial ライセンスは、**非商用の評価・研究・教育目的に限定して** 提供されま
す。その他の種別のライセンスは、発行時または別途の契約で個別に定める条件に
従います。

## 4. 個人につき 1 アカウント
個人は、同時に **1 つのみ** の QUBO++ ユーザーアカウントを保有・運用できま
す。以下の行為は禁止されます。

- メールエイリアス・別アドレスの使用、または同僚・親族の名義による登録を含
  む、複数アカウントの作成・運用（追加ライセンスの取得、その他アカウント
  単位の制限の回避を目的とするもの）
- 他者に対するアカウント認証情報の共有、またはアカウントの譲渡

当方は、当方の合理的な判断において本条に違反すると判断したアカウントを停止
または閉鎖し、紐づくライセンスを取り消すことができます。

## 5. 禁止事項
以下の行為は禁止されます。

- QUBO++ のバイナリ・ライブラリ・ライセンスキーの再配布またはサブライセンス
- バイナリのリバースエンジニアリング・逆コンパイル、またはソースコード導出
  の試み
- Trial ライセンスの下で QUBO++ を本番用途または商用目的に使用すること
- アクティベーション・マシンバインディング・検証・レート制限の各機構（特に
  非アクティベーション cooldown および更新ウィンドウを含むがこれに限られな
  い）の回避

## 6. 取得する情報
当方は、アカウント作成時にメールアドレス・氏名・所属・役職・国・利用目的を
取得します。また、ライセンスをアクティベートまたは検証する際に、マシン
フィンガープリント・ホスト名・OS・CPU/GPU 情報・RAM サイズ・IP アドレス・
ソフトウェアのバージョンを取得します。

利用者がユーザーポータルから任意でライセンスに付与したメモは、当該ライセ
ンスと共に保管され、QUBO++ 管理者が閲覧可能です。機微な個人情報や秘密
情報、第三者の機密情報を当該メモに入力しないでください。

利用者が作成したプログラム、当該プログラムが扱う問題インスタンス、および
計算結果については、当方は **取得・送信しません**。ライセンス検証通信には、
上記のメタデータのみが含まれます。

当方は、アカウント作成時に取得した情報（メールアドレス、氏名、所属、
役職、国、利用目的）を社内記録に用いるほか、有償 QUBO++ ライセンス
およびその関連製品に関する営業活動の目的で、当方の業務提携先（販売
代理店等を含む）に開示することがあります。ライセンス検証時に取得さ
れる情報（マシンフィンガープリント、IP アドレス、ハードウェア構成
等）は、これらの目的では開示されません。

利用者は、QUBO++ ウェブサイトに記載された問い合わせ窓口を通じて、
業務提携先への情報開示の停止をいつでも申し出ることができます。当該
停止は、利用者が現在保有するライセンスの稼働には影響しません。

## 7. 無保証
QUBO++ は **「現状有姿」** で提供され、商品性・特定目的への適合性・非侵害を
含むいかなる明示または黙示の保証もありません。

## 8. 終了
当方は、**理由の有無を問わず、また予告なく**、ライセンスを停止・取り消し、
またはアカウントを閉鎖できます。利用者はいつでも User Portal からアカウント
を削除できます。アカウント削除時には、紐づくすべてのライセンスは停止されま
す。

## 9. 規約の変更
当方は、本規約を随時更新することがあります。重要な変更については、メール
またはポータルへの掲示により通知します。
