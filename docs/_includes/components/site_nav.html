{%- comment -%} Override of just-the-docs site_nav. Filters site.html_pages by page.lang so the rendered navigation contains only pages in the current language (en or ja). This is critical for SEO: hreflang + canonical assume language- separated link graphs. Pages without a lang front-matter value (top-level redirect stubs) are excluded because they have nav_exclude: true already. {%- endcomment -%} {%- assign current_lang = include.lang | default: page.lang | default: 'en' -%}
