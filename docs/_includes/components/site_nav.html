{%- comment -%} Override of just-the-docs site_nav. Filters site.html_pages by both: 1. current language (page.lang: en or ja) — set by include.lang 2. current mode (cpp or python) — derived from page.python. Mode python: only /en/python/ or /ja/python/ items (page.python == true), plus pages flagged mode_shared: true. Mode cpp: everything else in the current language. This avoids the duplicate Installation/Quick Start/Case Studies entries that previously showed both /en/X and /en/python/X side-by-side. {%- endcomment -%} {%- assign current_lang = include.lang | default: page.lang | default: 'en' -%} {%- assign current_is_python = include.python | default: page.python -%}
